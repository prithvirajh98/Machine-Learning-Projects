	1. About the Project:
Every organisation has a HR department which deals with the various processes, like payroll process, conducting benefit analysis, hiring the right candidate for a particular role and so on. In this project we will be dealing with recruitment and placement process related to the company. Here in this project we have a huge dataset to extract the insights which might be useful for the HR Department to work and gain knowledge about the recruitment process of the market. There are many different insights that are extracted in the project which will be seen further in this project.

	2. About the Dataset:
The dataset is about the job details data for analytics industry. The companies in the dataset are Hong Kong based companies. Data consists of various job description for various analytics roles like Data Scientist, Data Analyst, Business Intelligence Analyst, Machine Learning Engineer and Data Engineer. The Dashboard made at the end will be made use of the HR Department.

The dataset has 6 separate dataset of different days which was combined to make a single dataset with many columns and attributes.

	3. Insights to be found:
		→ What are the total number of jobs available ?
		→ What are the total number of companies providing jobs ?
		→ What are the total jobs for various domains ?
		→ What are the various career level and it's distribution across various jobs ?
		→ What are the distribution of jobs across various analytics fields
		→ Which company is providing highest number of jobs ?
		→ Which company has the highest number of jobs ?
		→ What are the minimum required qualifications for the job roles ? 
		→ What are the various job types for different job titles ?
		→ Which are the TOP 5 companies with highest jobs ?
